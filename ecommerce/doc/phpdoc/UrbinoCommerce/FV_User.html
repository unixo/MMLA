Classe FV_User

L'oggetto FV_User rappresenta un utente contenuto nella base dati.

Esempio d'utilizzo della classe

  1.  require_once 'User.php';
  2.  
  3.  if (FV_User::validateUser('username', 'password')) {
  4.      $user =& $_SESSION['user'];
  5.      if ($user->isAdmin() === true) {
  6.          echo "Logged user is an administrator\n";
  7.      }
  8.      $user->basket->add(12);
  9.      $user->logout();
  10.  }
